Project description

This project is an auction marketplace where users can explore listings, create auction items, place bids, and view bidding history. I focused on building a clear user interface and improving usability.

Technologies: React, JavaScript, Tailwind CSS.

Improvements made: I refined the layout for better responsiveness, improved consistency across components, and cleaned up structure to make the project easier to maintain and present professionally.
